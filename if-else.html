Условни структури

На ова предавање ќе се фокусираме на Условните структури:

1. Условните структури се користат за проверка на одредени услови и извршување на соодветни делови од кодот во зависност од резултатот на условот. За ова повеќе ќе зборуваме во текот на денешниот час. Условни структури се: if-else, if-elif-else.

Структура:

if услов:

#код кој ќе се изврши ако условот е точен

elif услов:

#код кој ќе се изврши ако if условот не е точен

else услов:

#код кој ќе се изврши ако if и elif условите не се точни

Пример1:

x = 2
if x > 0:
print(“Бројот е позитивен”)

Објаснување: Во овој пример може да видиме дека променливата х е поголема од 0 и бидејќи условот е исполнет се печати “Бројот е позитивен”. Доколку условот не беше исполнет односно бројот беше негативен или еднаков на 0, немаше ништо да се испечати.

Пример2:

x = 2
if x > 0:
print(“Бројот е позитивен.”)
 elif x < 0:
print(“Бројот е негативен.”)
 else:
print(“Бројот е 0.”)

Објаснување: Во овој пример може да забележиме дека ако х е поголем од 0 се печати “Бројот е позитивен”, ако х е помал од 0 ќе се испечати “Бројот е негативен.”, a ако х е еднаков на 0 тогаш се печати “Бројот е 0.”




Пример3: Напиши програма во која преку тастатура ќе внесете два броја и ќе направете споредба. Резултатот да се испечати на екран:

x = input(“Внеси го првиот број”)
y = input(“Внеси го вториот број”)
if x > y:
print(“Бројот ”, x , “ е поголем од бројот ” , y)
 elif x < y:
print(“Бројот ”, y , “ е поголем од бројот ” , x)
 else:
print(“Бројот ”, x , “ и бројот ” , y , ” се еднакви.” )
