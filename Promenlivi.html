Променливи во Python и Влезно/Излезни функции

Откако инсталираме соодветна развојна околина за Python, веќе ќе научиме што се променливи како и каде се користат како и за влезно/излезните функции во Python.Пред да почнеме со пишување код да се запознаеме со неколку нови поими кои понатаму ќе ги употребуваме:

1. Варијабли/Променливи – Променливи или варијабли, се посебни локации во меморијата на компјутерот кои ги користиме за да чуваме вредност.
Постојат правила при пишување на имиња на променливи во Python:
- Името на променливата мора да започне со буква или знак на долна црта.
- Името на променливата не може да започне со број.
- Името на променливата може да содржи само алфа-нумерички знаци и долни црти (Az, 0-9 и _ ).
- Имињата на променливите се чувствителни на големи букви (age, Age и AGE се три различни променливи).
- Името на променливата не може да биде ниту еден од клучните зборови на Python.

Правилно се именуваат променливи: Неправилно се именуваат променливи:
myname = “Ivana” 1myname = “Ivana”
my_name = “Ivana’ my-name = “Ivana”
_my_name = “Ivana” my name = “Ivana”
myName = “Ivana”
MYNAME = “Ivana”
myname9 = “Ivana”

2. Податочни типови – Секоја променлива има свој податочен тип. Таа вредност може да биде цел број (int), децимален број (float), текст (string), буловa вредност (bool), листа (list) и друг тип на податок. Променливите во Python не мора да се декларираат сo точно одреден тип. Во моментот кога доделуваме вредност на променливата, Python автоматски го одредува типот на податокот кој го прима.

3. Коментари – Коментарите се делови од кодот кои не се извршуваат и служат за објаснување на функционалноста на кодот. Тие се означуваат со знакот ( # ) - тараба/хештаг и се игнорирани при извршување на програмата. Со ( # ) започува коментарот и го игнорира остатокот од редот/линијата. Доколку сакаме да додадеме коментар кој опфаќа повеќе од една линија код тогаш користиме ( """ ) – тројни наводници и коментарот го пишувате помеѓу парот од тројни наводници.
Пример: Како се користат коментари

X = 4 # x е од тип int (цел број)
y = 3.14 # y е од тип float (децимален број)
name = "Маја"; # name е од тип string (текст)
isTrue = true “”” isTrue е од типот bool (булова вредност) “””
List = [0,1,2,3] # List е од типот list (листа)

4. Casting – Casting во програмирањето се однесува на промена на типот од еден во друг податочен тип. Треба да се внимава при користење на casting бидејќи може да дојде до губење на информациите или да дојде до неправилни резултати. На пример: кога претвораме децимален број во цел број го губиме децималниот дел.
Еве неколку примери за casting:

x = int(2.6) “”” x ќе биде 2 – доаѓа до губење на децималниот дел 6 “””
y = float(3) # y ќе биде 3.0
z = str(8) # z ќе биде '8;
is_true = bool(0) # is_true ќе биде false
is_true = bool(1) # is_true ќе биде true

5. Функции – Влезно/Излезни функции се функции кои ја обезбедуваат комуникацијата помеќу програмата и корисникот или други делови од програмата. Овие функции овозможуваат внесување на податоци (влез) во програмата и печатење на информации од програмата (излез). Такви влезно/излезни функции се: input() и print(). За прикажување на податоци или вредности на стандарден влез (конзолата/терминалот) односно на екранот ја користиме функцијата print(). Може да прими една или повеќе вредности кои автоматски ги конвертира (претвора) во текст.
Пример1: Што ќе се испечати на екран?

x = 4 # x е од тип int
x = “Bambi” # x е од тип str
print(x) # за x ја печатиме последната вредност која му е доделена
